== README

This application is for merging pdf files. You can merge few files in one. You can merge a lot of files.
If you want to change order of merge you can drag and drop rows.

I used

* Ruby on Rails 4.1.0

* Haml

* Coffee-rails  4.0.0

* Deploy on Heroku

You can try this application on https://pdfmerger1.herokuapp.com/
