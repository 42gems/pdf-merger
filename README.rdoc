== README
= PdfMerge
This application is for merging pdf files.

Features of application:

* Merges few files in one. 
* Merge a lot of files.
* Change order of merge via drag and drop rows.
* Add or delete rows for adding.
* It merges only pdf files.

This application uses:

* Ruby on Rails 4.1.0

* Haml

* Coffee-rails  4.0.0

* Deploy on Heroku

You can try this {application}[https://pdfmerger1.herokuapp.com]
