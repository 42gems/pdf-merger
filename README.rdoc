= PdfMerge
This application is for merging pdf files.

Features of application:

* Merge a lot of pdf files in one.
* Changes order of merge via drag and drop rows.
* Changes the number of merging files.
* Validates file's format.

This application uses:

* Ruby on Rails 4.1.0

* Haml

* Coffee-rails  4.0.0

* Deploy on Heroku

You can try this {application}[https://pdfmerger1.herokuapp.com]
