== README
= PdfMerge
This application is for merging pdf files.

Features of application:

* You can merge few files in one. 
* You can merge a lot of files.
* If you want to change order of merge you can drag and drop rows.
* You can add or delete rows for adding.
* It merges only pdf files.

This application uses:

* Ruby on Rails 4.1.0

* Haml

* Coffee-rails  4.0.0

* Deploy on Heroku

You can try this {application}[https://pdfmerger1.herokuapp.com]
